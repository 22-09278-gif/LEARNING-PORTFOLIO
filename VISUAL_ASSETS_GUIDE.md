# Visuals Needed for the Learning Portfolio

Use these visuals to make your portfolio complete and credible:

## For each event page
- Event poster or invitation
- Program schedule / program flow
- Zoom screenshot or actual event photo
- Q&A screenshot or discussion proof
- Presentation slide screenshot
- Attendance sheet or participant proof
- Certificate of participation or appreciation, if available

## Recommended captions
- "This poster documents the official title, date, and program flow of the event."
- "This screenshot shows the online presentation proper during the seminar/colloquium."
- "This image documents audience engagement during the Q&A portion."
- "This slide shows the topic/project discussed and supports the learning reflection for this event."
- "This attendance proof supports my participation in the seminar/colloquium."

## Placement
- Use poster/program images as event page banners.
- Use Zoom and Q&A screenshots near the Evidence section.
- Use system diagrams or expected output images beside the event overview.
- Add a short caption under every image.
