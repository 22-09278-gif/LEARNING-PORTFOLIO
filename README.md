# EE 415 Learning Portfolio Multi-Page Website

This is a complete multi-page learning portfolio website. The navigation opens separate HTML pages instead of scrolling to sections.

## How to use
1. Open `index.html` in your browser to preview the site.
2. Edit the text in each HTML file if you want to personalize it further.
3. Replace the placeholder evidence instructions with your actual screenshots, certificates, and attendance proof.
4. Upload the files to GitHub Pages, Netlify, Vercel, or use the contents as references for Google Sites pages.

## Google Sites option
Google Sites does not support uploading a full multi-file website directly like GitHub Pages. If your instructor requires Google Sites, use this package as your content and visual guide:
- Create one Google Sites page per HTML page.
- Copy the text sections into each page.
- Upload the included event images as banners.
- Use the navigation menu in Google Sites to create the same page structure.
